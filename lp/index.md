## 満足度だけでなく、**業績向上**のための組織診断

「何を感じているか」に加え、「何をすれば業績が上がるか」まで。

満足度の把握にとどまらず、スタッフの態度・行動と業績の因果関係を3つのSTAGEで構造化。「何をすれば業績が上がるか」の優先順位を数値で示します。管理者はリアルタイムダッシュボードで全店を把握、経営層・人事部には因果分析レポートを提供します。

※ 個人へのフィードバックなし。店舗別・地域別・全社単位でのレポート

## Concept — 業績向上を3つのSTAGEで構造化する

満足度の把握にとどまらず、スタッフの満足感・積極性・チームの一体感が、どのように業績へ連鎖するかをモデル化。それぞれの要因に改善施策と優先順位をつけて提示します。

### STAGE 1｜組織活性化の源泉
スタッフ一人ひとりの仕事・職場・会社に対する満足感を測定

**キー指標：** 会社満足度 / 職務満足度 / 職務特性 / 職場環境 / 上司の関わり

### STAGE 2｜収益性向上エンゲージメント
理念・ビジョンへの共感と顧客視点意識を通じた収益貢献行動

**キー指標：** 理念・ビジョンへの共感 / 顧客視点意識 / 将来への期待

### STAGE 3｜チーム力と持続性
効果的なチームとしての協働と長期継続を生む組織基盤

**キー指標：** 効果的なチーム / 貢献意欲 / 勤続意思 / チームの基盤

> **3つのSTAGEが整うことで生まれるもの**
> 顧客対応型文化の醸成 → 生産性向上 → 業績拡大

## Problem — こんな課題はありませんか？

従来の満足度調査だけでは解決できない、フードサービス業ならではの悩みです。

### スコアは出るが、打ち手が分からない
満足度のスコアが出ても「どの要因が業績に効いているか」が見えず、全項目に施策を打とうとして疲弊する。

> 「スコアが低い項目を全部改善しようとしても、リソースが追いつかない」（外食チェーン 人事部）

### 問題が「店舗内」に矮小化される
会社全体としての支援不足や制度的な課題が見えにくく、すべてが「あの店長の問題」として処理され根本解決に至らない。

> 「店長への過度な負荷が改善されないまま、また離職が出た」（飲食チェーン エリアMG）

### 店舗間の差が感覚でしか語れない
好調店・不調店の違いを数値で説明できず、経営会議が感覚論になる。エリアMGによって対応にバラツキがある。

> 「数値で説明できないと、店長も納得してくれない」（ファストフード 教育担当）

## Two Values — 2つの価値を、異なる立場に

「今の状態をリアルタイムに把握する」管理者向けの機能と、「なぜそうなっているかを掘り下げる」経営・人事向けの機能を両輪で提供します。

### ダッシュボード｜For Managers / エリアMG・店長
リアルタイムで全店を把握。回答率・平均点・店舗ランキングをリアルタイムで確認。「今どの店が気になるか」を即座に判断できます。

- 有効回答数・回答率のリアルタイム表示
- TOTAL ENGAGEMENTスコアと3 STAGE表示
- 7指標×全店舗のヒートマップ一覧
- 強み・弱みファクター TOP3 / WORST3
- 前回比・全体平均比の差分表示
- 信号（青・黄・赤）による自動判定

### レポート｜For Executives & HR / 経営層・人事部
因果構造を経営判断に。人数×得点で全体スコアへの影響度を可視化。特に低いポイントを全体・店舗単位でアウトプット。施策の根拠になります。

- 全社・地域・店舗単位での出力
- STAGEごとの因果分析（β係数）
- 店長自己評価×スタッフ認識のギャップ分析
- 特に低スコアな要因の詳細抽出
- 経年推移グラフ（施策効果の検証）
- AIコメントによる自動解説（NotebookLM連携）

## Dashboard — 管理者ダッシュボード

### 01｜TOTAL ENGAGEMENTと3 STAGEをひと目で
有効回答数・回答率をリアルタイム表示。総合スコア（TOTAL ENGAGEMENT）と3つのSTAGEスコアを一画面で把握できます。Δ全体平均・Δ前回の差分も同時表示。改善・悪化を即座に確認できます。

> アンケート実施中から閲覧可能。**回収状況を見ながら督促のタイミングも判断できます**

### 02｜7つの指標を信号で即座に判定
7つのキー指標（会社満足度・職務満足度・将来への期待・顧客視点意識・貢献意欲・勤続意思・効果的チーム）を3段階の信号で自動判定します。

| 判定 | 条件 |
|------|------|
| 🟢 良好 | スコア ≧ 3.8 かつ ネガティブ < 10% |
| 🟡 注意 | どちらか一方のみ |
| 🔴 要改善 | スコア < 3.8 かつ ネガティブ ≧ 10% |

> 前回との差分・全体平均との差分も同時表示。**改善・悪化を一目で把握できます**

### 03｜全店舗×7指標をヒートマップで一覧
全店舗のスコアを色で視覚化。どの店舗のどの指標に問題があるか、スクロール一つで把握できます。事業所・役職・年代のフィルタリングに対応。「この年代層だけ低い」という細かい課題発見も可能です。

> **n<5はマスキング自動表示**。個人が特定されない設計で、スタッフが安心して本音を回答できます

### 04｜強み・弱みファクターを自動ランキング
18の要因から「最も得意なこと（STRONG FACTOR TOP3）」と「最も改善が必要なこと（WEAK FACTOR WORST3）」を自動で抽出します。全体平均比・前回比も同時表示。施策の効果測定にも活用できます。

> スコアと**ネガティブ回答率（回答「1」または「2」の比率）**を組み合わせて判定。スコアが高くても不満層が多ければ「注意」として検出します

### 05｜AIが診断結果を分析・解説
AIアドバイザーが診断結果を総合的に分析し、組織の強みと課題をわかりやすい言葉で解説。因果関係に基づく改善の方向性まで示唆します。データリテラシーに自信がない方でも、すぐにアクションに移せます。

## Report — 経営層・人事部向けレポート

「何が業績に効いているか」を因果関係で示す。意思決定の根拠になるアウトプット。

### 04｜全社サマリーと経年推移グラフ
総合平均の推移と3 STAGEスコアの推移を時系列グラフで表示。施策の前後で何が変化したかを定量的に把握できます。各レポートにはAI（NotebookLM）が生成した解説コメントが付属。数値を読み解く時間を省き、意思決定に集中できます。

> **経年比較**により、改善施策の効果検証が可能。「前回より〇〇がXポイント改善した理由は？」という議論が数字で行えます

### 05｜強み・弱みを全体・店舗単位で出力
18の要因から組織の強み（TOP）と改善が必要な要因（WORST）を自動抽出。人数×得点で全体スコアへの影響度も算出するため、「スコアは中程度でも人数が多く全体を下げている要因」を見逃しません。

> 全社・地域・店舗の各単位で出力可能。**同フォーマットで比較**できるため、店舗間の違いも把握できます

### 06｜STAGE分析で因果構造を把握
各STAGEのキー指標に、どの要因がどの程度影響しているかをβ係数で表示。「どこに手を打てば最も効果があるか」が数値で明確になります。人数×得点の積で全体スコアへの寄与度も算出。「スコアは高くないが人数が多いため全体を下げている要因」も特定できます。

### 07｜店長とスタッフのギャップを可視化
店長の自己評価とスタッフからの評価を比較するギャップ分析レポート。「店長は良くやっていると思っているが、スタッフには伝わっていない」という構造的な問題を検出します。

> **青：店長が高い項目**（意図が伝わっていないか、自己評価が高すぎる可能性）
> **赤：スタッフが高い項目**（スタッフが高い期待を寄せている）

## Difference — 従来の満足度調査との違い

| 比較項目 | 従来の満足度調査 | 本診断プログラム |
|----------|----------------|-----------------|
| **目的** | 不満要素の除去 | 業績向上に直結する行動要因の特定 |
| **アウトプット** | スコア（良い・悪いの判定のみ） | 優先施策・改善順位（β係数による影響度） |
| **対象の範囲** | 個人の不満・満足に焦点 | 個人＋チーム＋会社の支援を統合評価 |
| **店舗間比較** | スコアの大小比較のみ | 因果構造の違いまで比較可能 |
| **管理者の使い方** | 年1回の報告書として閲覧 | リアルタイムダッシュボードで常時モニタリング |
| **理論的背景** | なし（または独自定義） | 組織行動学（ロビンス理論）に基づく設計 |

## Theory — 学術理論に基づく設計

感覚ではなく、組織行動学の知見を診断モデルに落とし込んでいます。

### 「なぜ辞めるか」より「何をすれば業績が上がるか」を問う

スティーブン P.ロビンスの組織行動学では、従業員の**態度**が**行動**を生み、それが**業績**（生産性・離職率・貢献意欲）に直結することを体系化しています。

本診断はその理論構造を3つのSTAGEに落とし込み、「どの態度要因を改善すれば、どの行動が変わり、どう業績に影響するか」を数値で追える設計になっています。

```
態度（Attitude）
職務満足度・会社への満足・公平感・エンゲージメント
        ↓ 影響
行動（Behavior）
積極的な顧客対応・組織市民行動・チームへの貢献・勤続継続
        ↓ 結果
業績（Performance）
生産性向上・離職率低下・顧客満足・売上拡大
```

> 参照：S.P.ロビンス『組織行動のマネジメント』— 効果的なチームモデル・顧客対応型文化の変数・職務満足度と業績の関係性

## Target — こんな方におすすめ

### 経営者・役員
組織の現状を数値で把握し、投資優先度と施策の根拠を得たい

- 全社サマリーレポートで全体像を把握
- 経年推移で施策効果を検証
- 店舗間の因果構造の違いを比較

### 人事・教育担当
施策の優先順位を数値で立て、効果測定まで一貫して管理したい

- STAGE別因果分析で施策根拠を確立
- 対象者区分（店長・社員・PA）別の分析
- ギャップ分析で店長育成に活用

### エリアマネージャー
担当店舗の課題を数値で把握し、的確なサポートと説明を行いたい

- ダッシュボードで担当店を常時モニタリング
- ヒートマップで要注意店を即特定
- 信号判定で優先介入店を判断

## Flow — 最短1週間で診断スタート

1. **お問い合わせ** — フォームまたはデモ予約から。資料のみの請求も可能
2. **ヒアリング** — 店舗数・対象者・課題感をお伺い（30分程度）
3. **初期設定** — 組織構成・対象者リスト（CSV可）を登録
4. **診断スタート** — URLを配布してスタート。集計・レポートは自動生成

## FAQ — よくある質問

**Q. 満足度調査との一番の違いは何ですか？**
最大の違いは「目的」です。満足度調査は不満要素の除去を目的とするのに対し、本診断は業績向上に直結する行動要因を特定することを目的に設計されています。スコアの高低を見るだけでなく、「どの要因が業績（顧客満足・勤続意思・貢献行動）にどの程度影響しているか」をβ係数（パス係数）で示す点が大きく異なります。

**Q. 管理者はリアルタイムでどこまで見られますか？**
アンケート実施中からダッシュボードで確認できます。有効回答数・回答率・TOTAL ENGAGEMENTスコア・3 STAGEのスコア・7指標×全店舗のヒートマップ・強み弱みファクターがリアルタイムで更新されます。事業所別・役職別・年代別のフィルタリングにも対応しています。

**Q. レポートはどのような単位で出力されますか？**
全社・地域・店舗といった組織単位で出力できます。内容は①全体サマリー（推移グラフ）②STAGEごとの因果分析（β係数）③強み・弱みランキング④ギャップ分析（店長自己評価×スタッフ評価）⑤特に低スコアな要因の詳細抽出、が含まれます。個人へのフィードバックはありません。

**Q. 「業績向上」とは具体的に何を指しますか？**
本診断における業績向上とは、①生産性の向上②常習的欠勤・転職率の低下③顧客満足（顧客対応型文化）④組織市民行動（自発的な貢献行動）の4つです。これらはスタッフの職務満足度や態度が改善されることで、行動として現れる結果です（S.P.ロビンス組織行動学に基づく）。

**Q. 回答はどのくらいの時間がかかりますか？**
1人あたり10〜15分程度です。スマートフォンから回答できます。店長・社員・パートアルバイトそれぞれの立場に最適化された設問セット（一部設問が異なります）をご用意しています。

**Q. 匿名での回答はできますか？**
匿名回答に対応しています。また、5人未満のセグメントは自動マスキングされるため、個人が特定される心配がありません。スタッフが安心して本音を回答できる設計になっています。
