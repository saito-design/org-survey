# CSVエクスポート仕様

## 概要

組織診断の回答データを「横持ち形式（1行1回答者）」でエクスポートする。
この形式は統計分析（相関分析・重回帰分析）やクロス集計に最適。

## ファイル形式

- 形式: CSV（UTF-8 with BOM）
- ファイル名: `{会社名}_組織診断回答データ_{実施月}実施分.csv`
- 例: `株式会社サンプル様_組織診断回答データ_202602実施分.csv`

## ヘッダー構造（3行）

| 行 | 内容 | 例 |
|----|------|-----|
| 1行目 | 因子名（メタ列は空白） | `,,,,...,会社の満足度,職務満足度,...` |
| 2行目 | 設問文（メタ列は空白） | `,,,,...,私は総合的に会社に満足している,...` |
| 3行目 | 列名 | `実施月,本部コード,...,1,2,3,...` |

## 列構造

### メタデータ列（34列: 0-33）

| # | 列名 | データ元 | 説明 |
|---|------|---------|------|
| 0 | 実施月 | survey_id | 例: "26-Feb" |
| 1 | 本部コード | orgUnit.hq_code | 事業本部コード |

| 6 | 課名 | orgUnit.section | 課名 |
| 7 | 係コード | - | 係コード（未使用） |
| 8 | 係名 | - | 係名（未使用） |
| 9 | エリアコード | orgUnit.area_code | エリアコード |
| 10 | エリア名 | orgUnit.area | エリア名 |
| 11 | 業種コード | - | 業種コード（未使用） |
| 12 | 業種名 | - | 業種名（未使用） |
| 13 | 業態コード | orgUnit.business_type_code | 業態コード |
| 14 | 業態名 | orgUnit.business_type | 業態名 |
| 15 | 管理者コード | orgUnit.manager_code | 管理者コード |

| 33 | 適用終了日 | - | 未使用 |

### 回答データ列（65列: 34-98）

- 列34〜98: 設問1〜65の回答値（1-5）
- 1行目: 因子名
- 2行目: 設問文
- 3行目: 設問番号（1, 2, 3, ...）

## データ行

- 4行目以降: 各回答者のデータ
- 1行 = 1回答者
- 回答がない設問は空白

## 利用用途

1. **統計分析**: pandas等での相関分析・重回帰分析
2. **クロス集計**: 属性別の平均点比較
3. **セグメント抽出**: 条件に合致する回答者リストの作成
